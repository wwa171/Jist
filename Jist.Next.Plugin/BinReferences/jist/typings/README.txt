IMPORTANT NOTICE

This directory contains the TypeScript declaration (.d.ts) for IntelliSense
to work properly in Jist.Next.  Modifying any parts of these files may cause
your Jist installation to stop working, or report incorrect information.  Do
not attempt to modify or create any files in this directory unless you know
what you are doing.

Some declaration files in this directory are replaced every time Jist is
reloaded.

No support shall be provided for modified typings directories.
